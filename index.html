PRODUCT SCOPE

이 프로젝트에서 제가 다룬 범위

이 프로젝트에서 저는 랜딩 페이지 한두 개를 만든 것이 아니라, 실제 사용자가 쓰는 AI 제품 전체를 설계하고 구현했습니다. 메인 홈과 검색/채팅 UX, AI 도구 상세, 생산성 도구군, 커뮤니티, 프로필/설정, 플랜 결제 화면 같은 프론트엔드부터, 추천·생성·스트리밍·업로드·OCR·코드 분석 API, 구독/사용량/결제 정책, Google·Apple 인앱결제, 커뮤니티 신고/운영, MSSQL 기반 데이터 구조까지 서비스 전반을 직접 다룬 프로젝트입니다.

사용자 경험

  • 메인 홈, About, 정책/가이드, 온보딩, 앱 다운로드 유도까지 포함한 서비스 진입 경험 설계
  • AI 검색 페이지, AI 도구 목록/상세, 필터·태그·정렬·페이지네이션, 관리자 도구 등록 UI 구현
  • AI 종합 채팅과 검색 채팅의 이중 구조 설계, 스트리밍 메시지와 카드형 결과 UX 구현
  • 생산성 도구군 9종 구성: 요약, 레포트, 자기소개서, 면접 준비, 강의 녹음 노트, PPT 초안, 발표 대본, 코드 생성, 코드 리뷰
  • PPT 생성과 편집, 프로젝트 단위 코드 리뷰, 파일 업로드 기반 분석 UI 등 고난도 입력 경험 구현
  • 커뮤니티 목록/상세/글쓰기/수정, 가이드라인, 게시판 카테고리 흐름과 운영성 있는 UX 구현
  • 프로필 편집, 아바타, 계정 삭제, 설정, 플랜/구독 상태 확인 등 사용자 계정 경험 전반 구현

백엔드/운영

  • Next.js API Routes 기반으로 검색, 채팅, 생성, 업로드, 문서 처리, 코드 리뷰, 커뮤니티, 관리자 기능까지 서버 로직 전반 구현
  • AI 추천/검색 라우팅, 멀티모델 채팅, 스트리밍 응답, 구조화 카드 파싱, 도구 설명 생성 로직 설계
  • 이미지/영상 생성 API, 문서 OCR, 프레젠테이션 스크립트, 슬라이드 생성, 프로젝트 업로드/분석 파이프라인 구현
  • TossPayments 주문 생성, 정기결제, 수동 청구, 결제 상태 조회, 구독 취소/다운그레이드 등 결제 수명주기 구현
  • 사용량 제한과 리셋 정책을 서비스별로 설계하고, 이미지/영상/생산성 도구 한도 관리 로직 구현
  • 모바일 인증, 모바일 사용량 조회, Google IAP 검증, Apple IAP 검증/복구, App Store 서버 웹훅 처리 구현
  • MSSQL 연결 풀, 트랜잭션, 사용자/구독/결제/사용량/커뮤니티/신고 데이터 저장 구조 운영
  • 커뮤니티 신고, 공지 권한 제어, 관리자 기능, 운영용 API까지 포함한 서비스 운영 레이어 구현
